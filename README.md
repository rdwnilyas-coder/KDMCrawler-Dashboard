# KDMCrawler — Dashboard

Dashboard pemantauan media **Kang Dedi Mulyadi** (Gubernur Jawa Barat).
Sumber: portal berita Indonesia + YouTube. Periode **10 Mar – 10 Jun 2026**.

🔗 **Live:** https://rdwnilyas-coder.github.io/KDMCrawler-Dashboard/

## Isi
- **Berita** — 1.671 artikel dari 18 portal (timeline, per-portal, reporter, kanal, kata kunci judul)
- **YouTube** — 720 video, 511 jt views (timeline upload, kategori, channel, video terpopuler)
- **Komentar** — 206.894 komentar (top-liked, kata kunci, volume per video)

Data mentah dikumpulkan oleh [KDMCrawler](https://github.com/rdwnilyas-coder/KDMCrawler) (privat).
Dashboard ini *descriptive layer*; layer analitik (NER/sentiment/topik) menyusul.

_Static site — ECharts via CDN, data sebagai JSON._
